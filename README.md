# 购物网站示例

这是一个非常简易的跨境电商网站示例，用于展示从中国采购的商品。

## 入门指南

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm start
   ```

3. 在浏览器打开 `http://localhost:3000` 查看商品列表。

## 项目结构

- `server.js` – 提供 API 和静态文件的 Express 服务器。
- `public/index.html` – 从 API 获取商品数据并展示的前端页面。
- `data/products.json` – 示例商品数据。

此仓库仅作为入门演示，你可以在此基础上扩展购物车、用户系统以及支付等功能。
